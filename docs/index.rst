################################
Welcome to XGeo's documentation!
################################

XGeo is a geospatial extension to the xarray_ using  rasterio_ and geopandas_.  XGeo provides a convenient way to
manipulate geospatial data with a number of general day-to-day operations used in the geospatial world.

.. _xarray: http://xarray.pydata.org
.. _rasterio: https://rasterio.readthedocs.io
.. _geopandas: http://geopandas.org/
