Introduction¶

XGeo is a geospatial utility library which uses xarray, rasterio, pandas, geopandas to provide user geospatial functionality like transformation, re-projection, statistics, zonal statistics calculation, pixel sampling, vector based subsetting etc.

Git Repository¶

Source code of the library is found at https://github.com/Geosynopsis/xgeo

Note

If you clone the repository, the files in the test folders are not cloned. Please use git-lfs to pull those files.
